# GenericBoxTest1
C++ code to test various boxing strategies, version 1

The C++ file provides 

* a concept to give a simple abstract interface for boxes that may 
  contain immediate values or points, and
* timing tests for extracting tags and doing simple arithmetic.

Run the test using

    bash run.sh

After it is done, it will have generated the file 

    tag-test-results.tgz
